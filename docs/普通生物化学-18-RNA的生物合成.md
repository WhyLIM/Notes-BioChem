# 普通生物化学-18-RNA 的生物合成

## 转录

### 原核生物的转录

#### 基本过程

$$\ce{\begin{bmatrix}n1ATP\\
    n2GTP\\
    n3CTP\\
    n4UTP\\
\end{bmatrix} ->[\color{#58B1FF}{模板（RNA），Mg^{2+}}][\color{#F65353}{RNA 聚合酶}] RNA + (n1 + n2 + n3 + n4)PPi}$$

RNA 聚合酶与 DNA 模板的嘌呤核苷-5'-三磷酸结合成 1 个酶-DNA 复合物，DNA 构象改变，解旋，以其中 1 条链为模板合成 DNA。
